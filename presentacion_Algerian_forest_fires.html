01 — Descripción del Proyecto

Contexto y Objetivos

Este proyecto aplica técnicas de aprendizaje no supervisado sobre el dataset Algerian Forest Fires para identificar grupos naturales de condiciones climáticas y meteorológicas asociadas al riesgo de incendio forestal.

El dataset recopila observaciones diarias de dos regiones de Argelia (Bejaia y Sidi Bel-abbes) durante los meses de junio a septiembre de 2012, incluyendo índices del sistema canadiense FWI (Fire Weather Index).

# Clase cargador_datos — carga CSV y calcula info básica de calidad

import pandas as pd

class cargador_datos:

    def __init__(self, ruta_csv):
        self.ruta_csv = ruta_csv
        self.data = None
        self.filas = 0
        self.columnas = 0
        self.porcentaje_nulos = 0
        self.duplicados = 0

    def cargar_datos(self):
        self.data = pd.read_csv(self.ruta_csv)
        self.filas, self.columnas = self.data.shape

        total_celdas = self.filas * self.columnas
        total_nulos  = self.data.isnull().sum().sum()
        self.porcentaje_nulos = (total_nulos / total_celdas) * 100

        self.duplicados = self.data.duplicated().sum()
        self.tipos_variables = self.data.dtypes

    def resumen(self):
        print("Filas:", self.filas)
        print("Columnas:", self.columnas)
        print("Porcentaje de nulos:", round(self.porcentaje_nulos, 2), "%")
        print("Duplicados:", self.duplicados)
        print("Tipos de Variables:", self.tipos_variables)

04 — Proceso de Clustering

Selección del Número Óptimo de Clusters

Antes de aplicar K-Means, se estandarizaron las variables con StandardScaler para garantizar que todas contribuyan por igual al cálculo de distancias.

Escalamiento de Variables

Proceso_Cluster.ipynb — Escalamiento
from sklearn.preprocessing import StandardScaler
from sklearn.cluster import KMeans
from sklearn.metrics import silhouette_score

# Eliminar variables no relevantes para distancia euclidiana
df_cluster = df.drop(columns=["day", "month", "year", "Classes"])

# Normalización estándar (media=0, std=1)
scaler    = StandardScaler()
df_scaled = scaler.fit_transform(df_cluster)

Método del Codo

Se evaluaron valores de k de 1 a 9. La inercia (suma de distancias al cuadrado al centroide más cercano) se reduce drásticamente hasta k=3, donde la mejora se vuelve marginal.

Proceso_Cluster.ipynb — Método del Codo
inertia  = []

for k in k_values:

05 — Interpretación de Clusters

Segmentación Final de Riesgo de Incendio

Con k=3, el modelo K-Means clasifica cada observación en uno de tres grupos según las condiciones meteorológicas y los índices FWI. Cada cluster recibe una etiqueta de riesgo basada en la media de FWI.

Clusterización_Interpretacion.ipynb
# Features seleccionadas para clustering
features = ['Temperature', 'RH', 'Ws', 'Rain',
            'FFMC', 'DMC', 'ISI', 'BUI', 'FWI']
X = df[features]

scaler   = StandardScaler()
x_scaled = scaler.fit_transform(X)

# K-Means final con k=3
kmeans    = KMeans(n_clusters=3, random_state=42, n_init=10)
df['Cluster'] = kmeans.fit_predict(x_scaled)

# Asignar etiquetas por media de FWI
centros = df.groupby('Cluster')['FWI'].mean().sort_values()
mapeo   = {centros.index[0]: 'Bajo',
           centros.index[1]: 'Moderado',
           centros.index[2]: 'Alto'}
df['Riesgo'] = df['Cluster'].map(mapeo)

06 — Resultados y Conclusiones

Hallazgos Principales

01
El dataset tiene excelente calidad

Con 0% de valores nulos y 0 duplicados en 243 registros y 15 variables, el dataset no requirió imputación ni eliminación de registros, garantizando fiabilidad en los resultados.

02
K=3 es la partición más interpretable

Aunque el Silhouette Score máximo se obtuvo en k=2 (0.308), la partición en tres clusters ofrece mayor riqueza semántica, permitiendo distinguir condiciones de riesgo bajo, moderado y alto de forma accionable.

03
La temperatura es el predictor más discriminante

Los centroides muestran una diferencia de 7°C entre el cluster de bajo riesgo (28.3°C) y el de alto riesgo (35.4°C). Combinado con el FWI, temperatura y humedad son los factores dominantes en la segmentación.

04
El FWI amplifica la señal de riesgo 40x

El índice meteorológico de incendio (FWI) pasó de 0.44 en el grupo de bajo riesgo a 18.08 en el grupo de alto riesgo, evidenciando que las condiciones críticas se caracterizan por una acumulación simultánea de varios factores adversos.

05
Pipeline reproducible y modular

La arquitectura del proyecto — con la clase cargador_datos, notebooks independientes por etapa y el módulo de interpretación — facilita su reutilización sobre nuevos datos de otras regiones.
